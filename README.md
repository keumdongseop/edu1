# 회의실 예약 시스템 - 초보자용 교육 자료

풀스택 웹 개발을 배우는 가장 간단한 프로젝트입니다.

## 🚀 빠른 시작

```bash
npm install
npm run init-db
npm run dev
```

## 📁 파일 구조

```
classroom/
├── index.html     # 웹페이지 (HTML)
├── style.css      # 디자인 (CSS)
├── main.js        # 프론트엔드 (JavaScript)
├── server.js      # 백엔드 (Node.js)
└── bookings.db    # 데이터베이스 (SQLite)
```

## 📚 학습 내용

- **프론트엔드**: 사용자가 보는 화면
- **백엔드**: 데이터 처리 서버
- **데이터베이스**: 정보 저장소
- **API**: 프론트와 백엔드 통신
